# Yair Etziony


30 years of infrastructure, from VAX/VMS to the agentic era.
Berlin. Fond of Belgian beer and long walks.

[False Systems](https://www.false-systems.com). started as a joke.
Now it's a serious attempt to predict what infrastructure
looks like when agents are the operators.

I build tools that are transparent and easy to maintain.
Not because it's elegant, but because the next person
(human or agent) needs to understand what they're touching.

---

## False Systems

Infrastructure for agents as first-class operators.


- [Sykli](https://github.com/yairfalse/sykli) — CI without YAML
- [Kerto](https://github.com/false-systems/kerto) — context that survives the session
- [Nopea](https://github.com/false-systems/nopea) — CD that remembers what it deployed
- [Ahti](https://github.com/yairfalse/ahti) — causal graph for infrastructure state
- [Polku](https://github.com/false-systems/polku) — typed event routing for agents
- [Rauha](https://github.com/yairfalse/rauha) — safe boundaries for agents that act
- [Vaisto](https://github.com/false-systems/vaisto) — Scheme with Hindley-Milner, for the BEAM
